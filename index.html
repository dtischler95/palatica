Палатица

Ова страница се мора отворити преко сервера, не двокликом.

Per Doppelklick geöffnet (file://) blockiert der Browser die ES-Module, die Seite bliebe leer. Starte den lokalen Server und öffne die Adresse im Browser:

python serve.py → http://localhost:8080

Oder nutze die gehostete Version.
